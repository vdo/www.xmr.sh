A script to set up a new server with a supercharged Monero node.

Just run the following in your terminal,
then follow the onscreen instructions:

sudo bash -c "$(curl  -sLSf https://get.xmr.sh)"

Features

* Traefik with Let's Encrypt certificates.
* Debian, Ubuntu and Fedora supported.
* Tor hidden service.
* Grafana dashboard with all the metrics.
* CORS settings to use your node in web apps.
